The portfolio is a high-tech SPA (Single Page Application), designed in a deep 'cosmic' style utilizing cutting-edge frontend development techniques.

## üöÄ Key Features

* **Premium Animations:** Leveraged **GSAP** (GreenSock Animation Platform) to create smooth, cinematic transitions and sophisticated reveal effects.
* **Interactive Experience:** Features a dynamic interactive background with particles that respond to cursor movement (parallax effect).
* **Solid Architecture:** Developed with **React 18** using functional components and modern hooks (`useState`, `useMemo`, `useRef`) for clean, maintainable code.
* **Responsive & Adaptive:** Powered by **Bootstrap 5** and custom media queries, ensuring the interface looks flawless on everything from 320px screens to 4K displays.
* **Performance Optimized:** Highly optimized rendering logic following React best practices to ensure stability and lightning-fast load times.

## üõ† Tech Stack

* **Frontend Framework:** React (Vite-powered)
* **Animation Engine:** GSAP (GreenSock)
* **UI Framework:** Bootstrap 5 + Flexbox/Grid
* **Styling:** CSS3 with CSS Variables for dynamic theme management.
* **Linter:** ESLint (configured with React Compiler rules)

## üíª Installation & Setup

1.  Clone the repository:
    ```bash
    git clone [https://github.com/91Helen/portfolio-vite.git](https://github.com/91Helen/portfolio-vite.git)
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Launch the development server:
    ```bash
    npm run dev
    ```
